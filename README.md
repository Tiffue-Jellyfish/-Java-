# 🏢 办公室管理系统（Java + SQL Server）

这是我在大二下完成的课程项目，使用 **Java** 连接 **SQL Server (SSMS)**，实现了一个简单的办公室管理系统。  
包括员工管理、部门管理、登录验证等功能。

---

## 📌 项目特点

- 使用 **JDBC** 连接 SQL Server 数据库  
- 图形化界面（Java Swing）  
- 支持添加、删除、修改、查询办公室人员信息  
- 采用 MVC 思想进行基本分层  
- 多窗口界面切换  
- 提供简单登录系统  

---

## 🗂️ 项目结构
├── src/ # Java 源码
├── icons/ # UI 图标资源
├── LIB/ # 依赖库（如 JDBC 驱动）
├── out/ # 编译文件（可忽略）
└── OfficeManagement.iml # IDEA 工程配置文件

---

## 🛠️ 技术栈

- **Java SE**
- **Swing**（图形界面）
- **SQL Server / SSMS**
- **JDBC**
- **IntelliJ IDEA**

---

## 🚀 如何运行项目

1. 克隆仓库  
   ```bash
   git clone https://github.com/Tiffue-Jellyfish/-Java-.git
在 IntelliJ IDEA 中打开项目

导入 LIB/ 目录下所有 jar 包

修改数据库连接配置（在代码中配置你的 SSMS 地址、账号、密码）

运行 Main.java
